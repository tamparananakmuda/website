# Template Video Script TAM (HFBE Format)

Format: Hook - Body - Button - Engagement
Durasi: 30-60 detik
Platform: TikTok, Instagram Reels, YouTube Shorts

---

## Meta

```
Sumber artikel: [link artikel TAM]
Pillar: [pilih pillar]
POV Tag: [pilih POV]
Hook formula: [question/contrarian/statment/story]
```

---

## Struktur Script

### HOOK (0-3 detik)

Wajib berhenti scroll. Satu kalimat pertama yang membuat orang tidak bisa lanjut.

```
HOOK: [kalimat pertama, langsung ke inti]
```

Formula hook yang bekerja:
- Pertanyaan tidak nyaman: "Kenapa kamu produktif tapi tetap merasa tertinggal?"
- Statement kontra-narasi: "Sukses di usia 20-an versi internet itu bohong."
- Data mengejutkan: "90% bisnis gagal. Dan bukan karena bodoh."
- Tamparan langsung: "Berhenti menyalahkan orang tuamu."

### BODY (3-45 detik)

Satu insight dari artikel. Bukan rangkum seluruh artikel. Pilih angle paling tajam.

```
BODY:
[Point 1: tamparan/kenyataan]
[Point 2: penjelasan singkat kenapa]
[Point 3: solusi atau takeaway utama]
```

Aturan:
- 1 video = 1 insight. Jangan rangkum seluruh artikel.
- Bahasa spoken, bukan written. Singkat, langsung.
- Bukan lip-sync, bukan trend hopping. Talking head atau text-on-video.

### BUTTON (45-55 detik)

CTA yang natural, bukan dipaksakan.

```
BUTTON: [CTA ke artikel full / newsletter / follow]
```

Contoh:
- "Baca versi lengkapnya di tamparananakmuda.com"
- "Kalau ini nampar kamu, follow untuk yang lain"
- "Link di bio buat baca full article"

### ENGAGEMENT (55-60 detik)

Prompt untuk komentar/share. Opsional tapi recommended.

```
ENGAGEMENT: [pertanyaan untuk komentar]
```

Contoh:
- "Kamu pernah ngalamin ini? Comment di bawah."
- "Tag teman yang perlu baca ini."
- "Setuju atau tidak? Comment pendapatmu."

---

## Checklist

- [ ] Hook berhenti scroll dalam 3 detik pertama
- [ ] 1 insight saja, bukan rangkum artikel
- [ ] Bahasa spoken, bukan written
- [ ] CTA natural, bukan forceful
- [ ] Durasi 30-60 detik
- [ ] Caption/reply strategy siap
