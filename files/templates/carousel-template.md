# Template Carousel Instagram TAM

Format: 5-10 slides
Platform: Instagram, LinkedIn

---

## Meta

```
Sumber artikel: [link artikel TAM]
Pillar: [pilih pillar]
POV Tag: [pilih POV]
Jumlah slides: [5-10]
```

---

## Struktur Carousel

### Slide 1: Hook Cover

```
[Headline besar, kontras tinggi]
[Subheadline opsional, 1 baris]
[Brand mark TAM di pojok]
```

Aturan:
- Headline max 10 kata
- Font besar, bold, terbaca dari jauh
- Background solid atau subtle gradient (sesuai kategori color)

### Slide 2-3: Tamparan

```
[Statement kontroversial atau kontra-narasi]
[Data atau observasi pendukung]
```

### Slide 4-7: Penjelasan

```
[Point 1 dengan penjelasan singkat]
[Point 2 dengan penjelasan singkat]
[Point 3 dengan penjelasan singkat]
[Data atau contoh nyata]
```

Aturan:
- 1 point per slide, jangan padat
- Max 30 kata per slide
- Visual hierarchy: headline > body > footer

### Slide 8-9: Solusi

```
[Action item 1]
[Action item 2]
[Action item 3]
```

### Slide 10: CTA

```
[CTA ke artikel full]
"Baca versi lengkapnya di link bio"
[Brand mark TAM]
```

---

## Design Rules

- Background: OLED black (#0A0A0A) atau dark surface
- Accent color: sesuai kategori (lihat database categories.color)
- Font: Syne Bold untuk headline, sans-serif untuk body
- Minimal text, maksimal impact
- Konsisten dengan OG image aesthetic TAM
- No stock photos. Text-based atau custom graphics saja.

## Caption Strategy

```
[1 kalimat hook yang berbeda dari slide 1]
[2-3 kalimat konteks]
[CTA: "Swipe untuk selengkapnya →"]
[CTA: "Baca full di link bio"]
[Tags: 3-5 relevant hashtags]
```

## Checklist

- [ ] Slide 1 hook berhenti scroll
- [ ] Max 10 slides
- [ ] Max 30 kata per slide
- [ ] 1 point per slide
- [ ] CTA di slide terakhir
- [ ] Caption siap dengan CTA dan hashtags
- [ ] Visual konsisten dengan brand TAM
