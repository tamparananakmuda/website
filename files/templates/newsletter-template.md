# Template Newsletter TAM

Format: 400-600 kata
Frekuensi: 1x per minggu (Jumat)
Platform: Email via Resend

---

## Struktur Newsletter

### Subject Line

Max 50 karakter. Tidak clickbait. Tapi membuat penasaran.

```
Subject: [1 insight utama atau pertanyaan]
```

Contoh:
- "Kenapa kerja keras saja tidak cukup"
- "Gaji pertama: 3 hal yang saya sesali"
- "Kamu tidak kekurangan waktu. Kamu kebanyakan alasan."

### Header

```
TAMPARAN ANAK MUDA
Newsletter #[issue_number] | [tanggal]
```

### Opening (1 paragraf)

Personal, seperti surat dari teman yang jujur. Bukan formal.

```
[Halo pembaca],
[Konteks minggu ini: apa yang membuat saya ingin nulis ini]
```

### Insight Utama (2-3 paragraf)

Satu insight dari artikel minggu ini, atau observasi original.

```
[Statement utama]
[Penjelasan singkat]
[Contoh atau data]
```

### Rekomendasi (1 paragraf)

Satu hal yang bisa langsung dipraktikkan pembaca minggu ini.

```
[Minggu ini, coba: action item spesifik]
```

### Pertanyaan untuk Pembaca (1-2 kalimat)

Ajak pembaca reply email. Engagement > broadcast.

```
[Pertanyaan terkait insight tadi]
"Balas email ini, saya baca semua."
```

### Footer

```
---
TAMPARAN ANAK MUDA
Menyadarkan generasi muda akan kenyataan

Baca artikel terbaru: [link]
Dukung TAM: [link donasi]
Unsubscribe: [link]
```

---

## Aturan

- Bahasa lebih personal dari artikel website. Boleh casual: "gak", "kayak", "sih".
- Tidak duplikasi konten website. Newsletter bisa lebih personal, lebih reflektif.
- Max 600 kata. Newsletter bukan artikel. Pembaca baca di HP, di sela-sela waktu.
- Satu email = satu insight + satu rekomendasi + satu pertanyaan. Tidak lebih.
- Tidak ada em-dash. Gunakan koma atau titik.
- Tidak ada AI pattern (lengkap: lihat `files/HumanizerRules.md`)
- Tagline resmi di footer: "Menyadarkan generasi muda akan kenyataan"
